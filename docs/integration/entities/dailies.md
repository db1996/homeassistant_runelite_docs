# Dailies

Each daily activity has its own sensor.

## State

- `-1`: missing requirements  
- `0`: incomplete  
- `1`: complete  

## Entity ID

`sensor.runelite_%username_daily_%daily`

## Supported dailies

- Herb boxes  
- Battle staves  
- Essence  
- Sand  
- Flax  
- Arrows  
- Dynamite  

## Updates

- Updated on login by RuneLite plugin  
- Some update live during gameplay
