# Status Effects

Tracks active player status effects like poison or venom.

## Entity ID

`sensor.runelite_%username_status_effects`

## Attributes

`current_status_effects` (list):

- `name`: Poison / Venom  
- `number`: damage or boost value  
- `time`: reserved for future use
